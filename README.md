# QT + FFmpeg 5.x + x264 + x265 + SDL2 音视频播放器



使用了QT的QML设计界面,人机交互;



使用了FFmpeg 5.x  + x264 + x265 + SDL2 完成了音视频的解析到播放;



阅读了ffplay的源码,用到了ffplay的核心思想.



想熟悉ffmpeg和ffplay的朋友,都可以参考学习.
