# UPlayer
Video player

github:https://github.com/truedei/UPlayer

# 现有功能：
+ 1、切换背景图；
+ 2、播放视频（不能播放声音）；


# 计划支持但为完成的功能：
+ 1、视频的暂停功能；
+ 2、视频的开始播放功能；
+ 3、视频的停止播放功能；
+ 4、视频的快近功能；
+ 5、视频的快退功能；
+ 6、视频的倍速功能；
+ 7、视频的音频播放功能；
+ 8、视频的声音音量大小控制功能；
+ 9、视频的进度条显示和控制功能；
+ 10、视频的时间显示功能。
